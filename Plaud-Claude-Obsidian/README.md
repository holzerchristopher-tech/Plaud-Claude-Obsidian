# Plaud-Claude-Obsidian

## Overview
Plaud-Claude-Obsidian is a plugin designed to enhance the functionality of the Obsidian note-taking application. This plugin integrates with the Claude and Plaud APIs to provide users with advanced features and a seamless experience.

## Features
- **API Integration**: Interacts with Claude and Plaud APIs to fetch and manipulate data.
- **User Interface**: Provides modals and ribbons for easy interaction.
- **Custom Settings**: Allows users to configure preferences and options.

## Installation
1. Download the plugin files.
2. Place the `Plaud-Claude-Obsidian` folder into your Obsidian plugins directory.
3. Enable the plugin in the Obsidian settings.

## Usage
- Access the plugin features through the Obsidian command palette.
- Use the settings menu to customize your experience.
